# 🔐 Безопасность токенов FurChat

## Что было сделано:
✅ Твой Telegram токен **зашифрован** и хранится в коде как:
```
_ENCRYPTED_TG = "ENC:EhwTHBkdGR0fGxBra28dXhpGSV..."
```

## Как это работает:
1. В коде есть функция `_decrypt()`, которая расшифровывает токен
2. Токен расшифровывается **только во время работы программы**
3. В исходном коде видна только зашифрованная строка

## Проверка:
При запуске бота ты увидишь в логах:
```
✅ Токен успешно расшифрован (длина: 46)
```

## Если что-то пошло не так:
1. Проверь, что функция `_decrypt()` есть в файле
2. Убедись, что `_ENCRYPTED_TG` содержит правильное значение
3. В логах будет видно, если расшифровка не удалась
